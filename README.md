# aedes-nodered-server-ts

This projects implements an aedes MQTT-server, an express WebServer and a node-red instance.
It is programmed in TypeScript and uses nodemon as a dev-environment.

## Install

```
git clone https://github.com/gerrit-schramm/aedes-nodered-server-ts.git
npm i
```

## Run

```
npm run dev
```
